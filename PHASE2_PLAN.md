# Phase 2: 상세 페이지 디자인 통일 (Mobile-First)

## 🎯 목표
경매 상세 페이지를 홈 화면의 프리미엄하고 긴박한 디자인 톤앤매너와 통일하고, 모바일에서 한 손으로 조작하기 쉬운 UX로 개선합니다.

## 🛠 주요 변경 사항

### 1. 헤더/히어로 영역 개선
- **클럽 이미지**: `aspect-video`에서 `aspect-[4/3]` 또는 `aspect-square`를 고려하여 모바일에서 정보 가독성 확보.
- **오버레이 정보**: 이미지 하단에 클럽명, 테이블 타입, 지역 정보를 그라데이션 오버레이와 함께 배치.

### 2. 경매 현황 박스 프리미엄화
- **현재가 강조**: 홈 화면의 폰트 스타일(Black/Bold)을 적용하여 입찰 금액 시인성 극대화.
- **실시간 타이머**: `AuctionTimer`를 홈 화면의 `InlineTimer` 스타일과 긴급 기준(5분)을 공유하도록 업데이트.
- **섹션 재구성**: '경매 현황' 박스를 가장 상단으로 이동하여 실시간 긴박함 강조.

### 3. 입찰 히스토리 및 MD 정보
- **입찰 내역**: 카드 형태에서 리스트 형태로 간소화하여 스크롤 부담 감소.
- **MD 프로필**: 하이엔드 서비스 느낌을 주도록 정갈한 레이아웃으로 변경.

### 4. 하단 고정 입찰 패널 (Mobile-Only)
- **조작성 개선**: 빠른 입찰 버튼(프리셋)의 터치 영역 확대.
- **확인 절차**: `Sheet` 컴포넌트 내부의 금액 표시를 더 크게 강조.
